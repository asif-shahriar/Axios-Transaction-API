# Axios-Transaction-API
## Prerequisites
* Install Nodejs latest version
* Install VS code latest version
## How to run this project
* Clone the repo
* Open termianl in the root folder
* Give following commands:
```
npm install
```
```
npm start run .\Transaction.test.js
```
